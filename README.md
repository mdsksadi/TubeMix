# TubeMix
TubeMix is a fun and addictive color-sorting puzzle game built with Kotlin for Android.   
**Your goal is simple:** Sort the colored balls into their respective tubes until each tube contains balls of only one color. Sounds easy? Think again! With increasing difficulty levels and limited moves, TubeMix will challenge your logic and strategy skills.   
## Features
- Simple and intutive gameplay
- Beautiful color themes
- Multiple difficulty levels (Easy, Medium, Hard)
- Undo moves and restart levels
- Smooth animations with Jetpack Compose
- Online and offline play
## Tech Stack
- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Architecture:** MVVM
- **State Machine:** StateFlow
- **Minimum SDK:** 24
## How to Play
1. Tap a tube to select the top ball.
2. Tap another tube to move the ball (if the move is valid).
3. Sort all colors into separate tubes to win!
